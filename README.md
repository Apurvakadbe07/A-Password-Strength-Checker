# Password Strength Checker

## Overview

Password Strength Checker is a Machine Learning-based cybersecurity project developed using Python. The system evaluates password strength by analyzing important security factors such as password length, uppercase letters, lowercase letters, digits, and special characters. Based on these features, the model classifies passwords into Weak, Medium, or Strong categories.

## Objective

The primary objective of this project is to enhance password security awareness and help users create stronger passwords that are more resistant to unauthorized access and cyber threats.

## Features

* Password strength prediction using Machine Learning
* Classification into Weak, Medium, and Strong categories
* Automated feature extraction
* Data preprocessing and cleaning
* Model evaluation and performance analysis
* Real-time password checking

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook

## Machine Learning Model

This project uses the Random Forest Classifier to analyze password characteristics and accurately predict password strength. The model is trained on a password dataset and evaluated using various performance metrics.

## Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Train-Test Split
5. Model Training using Random Forest
6. Model Evaluation
7. Password Strength Prediction
